Central ESE Chat

The Matrix room #ese2020:matrix.ifsr.de is the central chat room you can use to communicate with fellow first semester students or ask tutors questions. You can join using your TU Dresden Matrix account by clicking the Join button below.

What is Matrix? Matrix is a distributed open-source messaging application that offers similar features like Slack (plus encrypted chats!). Using a single account, users can interact with and join rooms on different other servers. To learn more, visit matrix.org.
